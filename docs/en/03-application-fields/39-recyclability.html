Recyclability

Definition

可回收性是胶带废弃后能否被回收利用的能力。

Historical Background

早期胶带难以回收，造成大量垃圾。2000年代，可回收胶带、可降解胶带等环保产品问世，推动了循环经济。

Example

普通胶带用完就扔，而可回收胶带可以分离基材和胶层分别回收。可回收性就像"垃圾分类"，让资源循环利用。

Formula

回收率(%) = \frac{回收质量}{废弃质量} × 100\%

Core Concepts

可回收性是循环经济的基础——胶带由多种材料组成，实现回收需要材料易于分离。可回收性是胶带的"环保身份证"，决定了它的环境友好度。

Extended Applications

单一材料胶带：易回收（纸基胶带）；可分离胶带：基材可回收（BOPP胶带）；可降解胶带：自然分解（淀粉基胶带）；可堆肥胶带：工业堆肥（食品包装胶带）；化学回收胶带：原料回收（特种胶带）。
