Tape Thickness

Definition

胶带厚度是胶带的总厚度，包括基材和胶层，是影响性能和成本的关键参数。

Historical Background

早期胶带厚度无法精确控制，质量波动大。1970年代，β射线测厚仪问世，使厚度控制精度达到±1μm，胶带质量实现质的飞跃。

Example

同样一卷胶带，厚度增加0.01mm，一卷胶带的胶量就增加约10%，成本相应上升。但厚度太薄又会影响性能，所以要找到"甜蜜点"。胶带厚度就像衣服的厚薄，太厚闷热，太薄不保暖。

Formula

总厚度 = 基材厚度 + 上胶厚度 + 下胶厚度

Core Concepts

胶带厚度是性能-成本的平衡点。太厚浪费材料增加成本，太薄影响强度和耐久性。行业通常用±公差来衡量质量，如±5μm。厚度是胶带的"体重"，决定了它的能力边界。

Extended Applications

电子胶带要求厚度公差±1μm（影响贴合精度）；包装胶带要求厚度公差±10μm（影响成本核算）；汽车胶带要求厚度公差±5μm（影响装配间隙）；光学胶带要求厚度公差±0.5μm（影响光学性能）。
