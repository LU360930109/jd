Tape Thickness

Definition

Total thickness including substrate and adhesive layers, key for performance and cost.

Historical Background

1970s: beta-ray gauges achieved +/-1 um precision.

Example

+0.01 mm adds ~10% adhesive and cost. Like clothing: too thick overheats, too thin provides no warmth.

Formula

Total Thickness = Substrate + Top Adhesive + Bottom Adhesive

Core Concepts

Performance-cost balance point, measured via tolerance like +/-5 um.

Extended Applications

Electronic: +/-1 um; packaging: +/-10; automotive: +/-5; optical: +/-0.5 um.
