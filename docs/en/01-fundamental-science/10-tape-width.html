Tape Width

Details

Tape width is achieved by slitting wide parent rolls into required specifications. Common width specifications include 12mm, 18mm, 24mm, 36mm, 48mm, 60mm, 72mm, etc. Slitting precision and edge quality must be controlled to prevent burrs and delamination.
