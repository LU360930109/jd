Base Materials

Definition

基材是承载粘合剂的载体材料，决定胶带的物理性能和适用场景。

Historical Background

最早使用棉布和纸张作为基材。1950年代，聚丙烯（BOPP）薄膜的发明彻底改变了行业，使胶带成本降低50%，轻薄透明的胶带开始走进千家万户。

Example

同样粘贴一个纸箱，用布基胶带和透明胶带效果完全不同。布基胶带的棉布基材提供高抗拉强度，能承受20kg拉力；透明胶带的BOPP基材较薄，只能承受5kg。基材就像胶带的"骨架"，决定了它的强度上限。

Formula

胶带强度 = 基材强度 × 安全系数

Core Concepts

基材选择是一场平衡艺术：要轻薄还是要强度？要透明还是要耐温？要便宜还是要环保？没有完美的基材，只有最适合场景的基材。基材是胶带的"性格底色"，决定了它能走多远、能扛多重。

Extended Applications

BOPP基材占市场60%，用于普通包装；PET基材耐温150℃，用于电子；布基基材抗撕裂，用于地毯接缝；纸基基材可降解，用于环保包装；泡棉基材可填充缝隙，用于汽车密封。
