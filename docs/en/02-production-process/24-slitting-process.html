Slitting Process

Definition

分切工艺是将大卷胶带分切成小卷或特定宽度的加工工艺。

Historical Background

早期胶带宽度单一，使用不便。1960年代，高速分切机问世，使胶带可以被切割成任意宽度，极大拓展了应用场景。

Example

工厂生产出1000mm宽的大卷胶带，再分切成48mm、72mm等不同宽度的成品。分切就像"切蛋糕"，把大块变成小块。分切要保证边缘整齐、无溢胶、无脱层。

Formula

分切数量 = \frac{母卷宽度}{成品宽度}

Core Concepts

分切是规格定制的环节——要保证分切后的边缘整齐、无溢胶、无脱层。分切刀的锋利度、张力控制是关键。分切是胶带的"精雕细琢"，决定了产品的外观质量。

Extended Applications

窄带分切：1-6mm（电子胶带）；标准分切：10-100mm（包装胶带）；宽带分切：100-500mm（工业胶带）；超宽分切：500mm以上（建筑胶带）；精密分切：±0.1mm（光学胶带）。
