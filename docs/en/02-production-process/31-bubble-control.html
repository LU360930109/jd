Bubble Control

Definition

Eliminating or reducing bubbles during production.

Historical Background

1970s: vacuum and ultrasonic defoaming technologies introduced.

Example

Bubbles in screen protector adhesive create rainbow patterns.

Formula

Bubble Rate (%) = Bubble Area / Total Area x 100%

Core Concepts

Optical defect root cause: scatter light, increase haze, reduce transparency.

Extended Applications

Ordinary: <5%; optical: <1%; display: <0.5%; lens: <0.1%; aerospace: <0.01%.
